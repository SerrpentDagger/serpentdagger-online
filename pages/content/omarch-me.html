Omarch-me

Omarch-me is a fork of Omarchy by DHH: "a beautiful, modern & opinionated Linux distribution". This fork makes minor modifications to the install scripts that prioritise open-source apps and allow you to choose yourself which default apps to install with the OS.

Omarch-me is exactly the same as Omarchy after the initial installation. During install, however, the user will be asked if they want all the default Omarchy apps, webbaps, and potentially the proprietary Nvidia drivers (if applicable).

If the user chooses to customise their installation, only basic system apps will be automatically installed. The user then chooses extra apps to install (with their Omarchy configurations) from several small lists: 'Extra System', 'Media/Communications', 'Developer', and 'Unfree'.

From there, you can enjoy your customised Omarchy setup without the artefacts of unwanted apps!
